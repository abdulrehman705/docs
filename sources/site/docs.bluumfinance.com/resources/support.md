Support channels

| Channel | Contact | Best for |
| --- | --- | --- |
| **Email** | `support@bluum.finance` | Credentials, access, compliance, general questions |
| **Developer support** | `ope@bluumfinance.com` | Integration help, technical questions, API issues |
| **Emergency trading desk** | `+1 (415) 555-0112` | Urgent production issues during market hours (24/5) |

## 

[​](https://docs.bluumfinance.com/#when-contacting-support)

When contacting support

Include the following for faster resolution:

- **Your company name** and dashboard account email
- **Environment** (sandbox or production)
- **Request ID** (`X-Request-Id` header from the API response)
- **Error code** (e.g., `BLUM-400-001`)
- **Timestamp** of the issue
- **Steps to reproduce** (API calls made, parameters used)

## 

[​](https://docs.bluumfinance.com/#response-times)

Response times

| Severity | Description | Target response |
| --- | --- | --- |
| **Critical** | Production trading or funding blocked | 1 hour (during market hours) |
| **High** | Feature degraded, workaround available | 4 hours |
| **Medium** | Non-blocking issue or question | 1 business day |
| **Low** | Feature request, documentation improvement | 3 business days |

## 

[​](https://docs.bluumfinance.com/#requesting-production-access)

Requesting production access

Email `support@bluum.finance` with:

1. Company name and dashboard email
2. Integration model (Self-Directed or Wealth Management)
3. Confirmation of sandbox testing completion

See [Going Live](https://docs.bluumfinance.com/get-started/go-live) for the full production readiness checklist.
